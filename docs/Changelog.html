5.0.0

In this release we defined a whole new API that does not rely on promises anymore. To migrate please also update RxJS to a version greater than 6.

Decorator

We dropped support for the decorator syntax as it added a lot of testing surface. Please use libraries like recompose with componentFromStream to fill the gap. If you need help with this feel free to open an issue and we will provide more context and guidance.
